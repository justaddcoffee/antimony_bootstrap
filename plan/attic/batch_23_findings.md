## 2. Key Pathways Identified

### 2.1 Amyloid-Beta Production, Processing, and Clearance
**Papers**: PMC3568930, PMC3570694, PMC3604133, PMC3618974, PMC3623032, PMC3651582, PMC3653296, PMC3661739

Key findings for ODE modeling:
- **APP processing**: PMC3623032 describes distinct alpha/beta/gamma-secretase cleavage patterns, providing basis for competing pathway ODE reactions
- **AB isoform distribution**: PMC3568930 quantifies AB42/AB40/pGluAB3-42 ratios in FAD vs SAD brains -- useful for parameter estimation of relative production rates
- **AB clearance via ApoE**: PMC3651582 demonstrates ApoE-directed therapeutics clear AB rapidly -- provides clearance rate data
- **Microglial AB clearance**: PMC3653296 quantifies microglial phagocytic uptake and degradation of AB
- **AB oligomer concentrations**: PMC3604133 and PMC3618974 provide measured oligomer concentrations in brain, CSF, and plasma

### 2.2 Tau Phosphorylation, Aggregation, and Propagation
**Papers**: PMC3575082, PMC3601591, PMC3615658, PMC3623298, PMC3648341, PMC3665718, PMC3692283

Key findings for ODE modeling:
- **Tau seeding kinetics**: PMC3575082 demonstrates synthetic tau fibrils seed NFTs with measurable time courses -- critical for prion-like propagation module
- **Activity-dependent tau release**: PMC3615658 shows neuronal activity stimulates tau secretion -- provides tau release rate data for ISF compartment
- **Cis/trans p-tau isomerization**: PMC3601591 describes Pin1-mediated conversion between cis (pathogenic) and trans (normal) p-tau conformers -- novel ODE reaction
- **Network diffusion model**: PMC3623298 provides explicit diffusion equations for disease spreading across brain networks -- directly translatable to ODE system
- **Seeded aggregation**: PMC3665718 reviews nucleation-dependent polymerization kinetics applicable to tau and AB

### 2.3 Neuroinflammation and Microglial Function
**Papers**: PMC3620049, PMC3631573, PMC3632747, PMC3653290, PMC3653296, PMC3677161, PMC3691612

Key findings for ODE modeling:
- **Microglial phagocytosis decline**: PMC3620049 shows functional impairment of microglia coincides with increasing AB deposition -- provides dose-response relationship for phagocytosis rate as function of AB load
- **Microglial activation states**: PMC3632747 reviews factors regulating microglial activation including cytokine cascades -- basis for multi-state microglia module
- **TREM2/TYROBP signaling**: PMC3677161 identifies TYROBP/TREM2 network as key immune hub -- provides network topology for signaling module
- **Cytokine production**: PMC3653290 reviews inflammatory mediator release rates by activated microglia

### 2.4 Autophagy/Lysosomal Pathway
**Papers**: PMC3638014, PMC3647462

Key findings for ODE modeling:
- **PS1-dependent lysosomal function**: PMC3647462 demonstrates presenilin 1 is required for lysosomal acidification and autophagy -- PS1 mutations impair protein degradation rates
- **TFEB regulation**: PMC3638014 describes TFEB master regulation of autophagy/lysosome biogenesis -- provides transcription factor-dependent regulation for degradation rates

### 2.5 ApoE Metabolism and Transport
**Papers**: PMC3640999, PMC3651582, PMC3661739, PMC3683865

Key findings for ODE modeling:
- **ApoE ISF turnover**: PMC3640999 provides direct in vivo measurements of ApoE production and clearance in brain ISF -- turnover half-life data directly usable as parameters
- **ApoE-AB interaction**: PMC3651582 shows ApoE-targeted interventions clear AB -- quantitative clearance data

### 2.6 Biomarker Dynamics and Disease Progression
**Papers**: PMC3595157, PMC3600112, PMC3622225, PMC3656503, PMC3687777

Key findings for ODE modeling:
- **Biomarker trajectory shapes**: PMC3595157 fits sigmoid curves to 5 major AD biomarkers -- provides temporal shape parameters for model validation
- **Updated biomarker cascade**: PMC3622225 updates the Jack et al. hypothetical model -- temporal ordering constraints for multi-compartment dynamics
- **AB-neurodegeneration coupling**: PMC3600112 quantifies covariance between AB deposition (PiB), glucose metabolism (FDG), and gray matter volume

### 2.7 Synaptic Dysfunction
**Papers**: PMC3570694, PMC3592200, PMC3637871

Key findings for ODE modeling:
- **AB oligomer synaptic toxicity**: PMC3570694 demonstrates AB from AD brains impairs LTP and synaptic remodeling -- dose-response data for synaptic module
- **Activity-dependent DNA damage**: PMC3637871 shows AB exacerbates neuronal DSBs via NMDA receptors -- extrasynaptic NMDA activation pathway

## 3. Extractable Reactions and Rate Laws

### 3.1 APP Processing Module

```antimony
// Alpha-secretase cleavage (non-amyloidogenic)
// PMC3623032
J_alpha: APP_Neuron -> sAPPalpha_BrainISF + CTF83_Neuron; k_alpha * APP_Neuron * V_Neuron

// Beta-secretase cleavage (amyloidogenic)
// PMC3623032
J_beta: APP_Neuron -> sAPPbeta_BrainISF + CTF99_Neuron; k_beta * APP_Neuron * V_Neuron

// Gamma-secretase cleavage of CTF99 producing AB42 vs AB40
// PMC3568930 (isoform ratios), PMC3623032
J_gamma42: CTF99_Neuron -> AB42_BrainISF + AICD_Neuron; k_gamma42 * CTF99_Neuron * V_Neuron
J_gamma40: CTF99_Neuron -> AB40_BrainISF + AICD_Neuron; k_gamma40 * CTF99_Neuron * V_Neuron

// Pyroglutamate modification
// PMC3568930
J_pGlu: AB42_BrainISF -> pGluAB342_BrainISF; k_pGlu * AB42_BrainISF * V_BrainISF
```

### 3.2 AB Oligomerization and Aggregation

```antimony
// AB42 oligomerization (nucleation-dependent)
// PMC3604133, PMC3618974, PMC3665718
J_olig: 2 AB42_BrainISF -> AB42olig_BrainISF; k_olig * AB42_BrainISF^2 * V_BrainISF

// AB oligomer to fibril (seeded polymerization)
// PMC3665718
J_fib: AB42olig_BrainISF + AB42_BrainISF -> AB42fib_BrainECF; k_fib * AB42olig_BrainISF * AB42_BrainISF * V_BrainISF

// AB plaque deposition
// PMC3600112
J_plaque: AB42fib_BrainECF -> AB42plaque_BrainECF; k_dep * AB42fib_BrainECF * V_BrainECF
```

### 3.3 AB Clearance Pathways

```antimony
// Microglial phagocytic clearance (declines with AB load)
// PMC3620049, PMC3653296
J_mg_clear: AB42_BrainISF -> ; k_mg_phag * MicrogliaActive_Brain * AB42_BrainISF / (Km_phag + AB42_BrainISF) * V_BrainISF

// Phagocytosis rate inhibition by AB load (negative feedback)
// PMC3620049
// k_mg_phag_effective = k_mg_phag_max * Ki_AB / (Ki_AB + AB42plaque_BrainECF)

// ApoE-mediated clearance
// PMC3651582, PMC3640999
J_apoe_clear: AB42_BrainISF + ApoE_BrainISF -> ApoEAB_BrainISF; k_apoe_bind * AB42_BrainISF * ApoE_BrainISF * V_BrainISF
J_apoe_degrad: ApoEAB_BrainISF -> ApoE_BrainISF; k_apoe_degrad * ApoEAB_BrainISF * V_BrainISF

// ISF bulk flow clearance
// PMC3640999
J_isf_clear: AB42_BrainISF -> AB42_CSF; k_isf * AB42_BrainISF * V_BrainISF
```

### 3.4 Tau Dynamics Module

```antimony
// Activity-dependent tau release from neurons to ISF
// PMC3615658
J_tau_release: Tau_Neuron -> Tau_BrainISF; k_tau_rel * NeuronalActivity * Tau_Neuron * V_Neuron

// Tau phosphorylation
// PMC3601591
J_tau_phos: Tau_Neuron -> pTau_Neuron; k_phos * Tau_Neuron * Kinase_Neuron * V_Neuron

// Cis-trans isomerization of p-tau (Pin1-mediated)
// PMC3601591
J_cis_trans: cisPTau_Neuron -> transPTau_Neuron; k_pin1 * Pin1_Neuron * cisPTau_Neuron * V_Neuron
J_trans_cis: transPTau_Neuron -> cisPTau_Neuron; k_spontaneous * transPTau_Neuron * V_Neuron

// Tau seeding/propagation (prion-like)
// PMC3575082, PMC3648341
J_tau_seed: TauSeed_BrainISF + Tau_Neuron -> 2 TauSeed_BrainISF; k_seed * TauSeed_BrainISF * Tau_Neuron * V_Neuron

// Network diffusion of tau pathology
// PMC3623298 -- diffusion model: dTau_i/dt = -beta * L_ij * Tau_j
// (Implemented as inter-compartment flows governed by connectivity matrix)
J_tau_diff_ij: Tau_Region_i -> Tau_Region_j; D_tau * connectivity_ij * (Tau_Region_i - Tau_Region_j)
```

### 3.5 Microglial Activation Module

```antimony
// Microglial activation by AB
// PMC3632747, PMC3653290
J_mg_act: MicrogliaResting_Brain -> MicrogliaActive_Brain; k_act * AB42_BrainISF * MicrogliaResting_Brain * V_Brain

// Cytokine release by activated microglia
// PMC3653290
J_tnf: -> TNFalpha_BrainISF; k_tnf * MicrogliaActive_Brain * V_Brain
J_il1b: -> IL1beta_BrainISF; k_il1b * MicrogliaActive_Brain * V_Brain
J_il6: -> IL6_BrainISF; k_il6 * MicrogliaActive_Brain * V_Brain

// TREM2-mediated signaling enhancement
// PMC3631573, PMC3677161, PMC3691612
J_trem2: TREM2_Microglia + DAM_ligand_BrainISF -> TREM2active_Microglia; k_trem2 * TREM2_Microglia * DAM_ligand_BrainISF
```

### 3.6 Autophagy/Lysosomal Module

```antimony
// TFEB activation and autophagy induction
// PMC3638014
J_tfeb_act: TFEB_cytoplasm_Neuron -> TFEB_nucleus_Neuron; k_tfeb * stress_signal * V_Neuron
J_autophagy: -> Autophagosome_Neuron; k_auto * TFEB_nucleus_Neuron * V_Neuron

// PS1-dependent lysosomal acidification
// PMC3647462
J_lyso_acid: -> LysosomalAcidity_Neuron; k_acid * PS1_Neuron * V_Neuron

// Lysosomal protein degradation (PS1-dependent)
// PMC3647462
J_lyso_degrad: ProteinAggregate_Neuron -> ; k_lyso * LysosomalAcidity_Neuron * ProteinAggregate_Neuron * V_Neuron
```

### 3.7 ApoE Metabolism Module

```antimony
// ApoE production in brain (astrocytes primary source)
// PMC3640999
J_apoe_prod: -> ApoE_BrainISF; k_apoe_prod * Astrocyte_Brain * V_Brain

// ApoE clearance from ISF
// PMC3640999 -- measured half-life ~14-16 hours in vivo
J_apoe_clear: ApoE_BrainISF -> ; k_apoe_clear * ApoE_BrainISF * V_BrainISF
// k_apoe_clear = ln(2) / t_half = approx 0.045 hr^-1

// ApoE lipidation
// PMC3651582
J_apoe_lipid: ApoE_BrainISF -> ApoElipid_BrainISF; k_lipid * ABCA1_Astrocyte * ApoE_BrainISF * V_BrainISF
```

### 3.8 Synaptic Dysfunction Module

```antimony
// AB oligomer binding to synapses
// PMC3570694
J_syn_bind: AB42olig_BrainISF + Synapse_Neuron -> AB42olig_Synapse; k_syn_bind * AB42olig_BrainISF * Synapse_Neuron

// PMC3570694
// LTP_efficiency = LTP_max * Ki_syn / (Ki_syn + AB42olig_Synapse)
```
