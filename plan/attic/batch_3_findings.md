## 3. Extractable Reactions and Rate Laws

### 3.1 APP Processing and Abeta Production

**Relevant papers:** PMC3192364, PMC3073454, PMC3169340, PMC3225957, PMC3273469, PMC3274732, PMC3189500

**Extractable reactions:**
```antimony
// APP cleavage by beta-secretase (BACE1)
J_BACE1_cleavage: APP_Neuron -> sAPPbeta_Neuron + CTFbeta_Neuron; k_BACE1 * APP_Neuron * BACE1_Neuron

// CTFbeta cleavage by gamma-secretase
J_gamma_cleavage: CTFbeta_Neuron -> AB40_BrainISF + AICD_Neuron; k_gamma * CTFbeta_Neuron * PS1_Neuron
J_gamma_cleavage_42: CTFbeta_Neuron -> AB42_BrainISF + AICD_Neuron; k_gamma42 * CTFbeta_Neuron * PS1_Neuron

// APP cleavage by alpha-secretase (non-amyloidogenic)
J_alpha_cleavage: APP_Neuron -> sAPPalpha_Neuron + CTFalpha_Neuron; k_alpha * APP_Neuron
```
**Rate type:** MA (mass action) or custom_conc_per_time
**Key parameters:** k_BACE1, k_gamma, k_gamma42, k_alpha
**Evidence:** PMC3273469 (BACE1 inhibition CSF pattern), PMC3169340 (metalloprotease/gamma-secretase regulation)

### 3.2 Abeta Clearance and Degradation

**Relevant papers:** PMC3192364, PMC3073454, PMC3257514

**Extractable reactions:**
```antimony
// Abeta clearance from brain ISF
J_AB_clearance_ISF: AB42_BrainISF -> ; k_clear_ISF * AB42_BrainISF
// ApoE-mediated Abeta clearance (isoform-dependent)
J_ApoE_clearance: AB42_BrainISF -> ; k_ApoE_clear * AB42_BrainISF * ApoE_BrainISF / (Km_ApoE + AB42_BrainISF)
// Abeta transport across BBB

J_AB_BBB_influx: AB42_Plasma -> AB42_BrainISF; k_BBB_influx * AB42_Plasma
```

**Key parameters:** k_clear_ISF, k_ApoE_clear (ApoE2 > ApoE3 > ApoE4), k_BBB_efflux, k_BBB_influx
**Evidence:** PMC3192364 (ApoE isoform-dependent clearance), PMC3073454 (decreased CNS Abeta clearance in AD)
