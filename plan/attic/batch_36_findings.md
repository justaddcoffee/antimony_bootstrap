## 2. Key Pathways Identified

### A. Tau Metabolism and Phosphorylation
- **Tau phosphorylation/dephosphorylation cycle** at specific sites (T181, S199, S202, S214, T217, T231, S404)
- **Tau turnover**: intracellular half-life vs extracellular (ISF/CSF) half-life of 1-2 hours
- **Tau aggregation**: site-specific aggregation t1/2 ~7 hours; oligomer -> fibril -> NFT progression
- **Tau fragmentation**: generation of distinct fragments in CSF vs brain
- **Tau spreading**: prion-like seeding and network-based propagation
- **Sleep-wake regulation**: neuronal activity drives ISF tau release

### B. Amyloid-beta Processing and Clearance
- **APP processing**: BACE1 and gamma-secretase cleavage in endosomal compartment
- **Abeta production flux**: endosomal pH-dependent BACE1 activity
- **Abeta clearance**: microglial phagocytosis (CD22-regulated), ISF drainage, BBB transport
- **Abeta aggregation**: monomer -> oligomer -> fibril -> plaque cascade
- **Cholesterol regulation**: independent modulation of Abeta production

### C. Neuroinflammation / Innate Immunity
- **Microglial activation states**: homeostatic -> DAM (disease-associated microglia) transition
- **TREM2 signaling**: soluble TREM2 modulates phagocytosis and cytokine release
- **ApoE-C1q complement interaction**: KD ~500 pM; complement cascade attenuation
- **Cytokine cascades**: IL-1beta, IL-6, TNF-alpha production and signaling
- **CD22-mediated phagocytosis inhibition**: age-dependent decline
- **CSF1R-dependent microglial survival**: depletion affects plaque dynamics

### D. Synaptic and Neuronal Function
- **Glutamate excitotoxicity**: Abeta-dependent NMDA/AMPA hyperactivation
- **Calcium dysregulation**: Ca2+ influx -> mitochondrial dysfunction
- **Cholinergic deficit**: acetylcholine signaling decline
- **LTP/LTD imbalance**: synaptic plasticity disruption
- **GABA-ergic inhibition loss**: disinhibition circuit

### E. Vascular and BBB Mechanisms
- **Capillary stalling**: neutrophil adhesion with half-life < 5 min; CBF reduction
- **BBB permeability**: VCAM1-mediated immune cell transmigration
- **ISF/CSF clearance**: glymphatic-like drainage of tau and Abeta

### F. Lipid/Cholesterol Metabolism
- **Cholesterol-tau axis**: independent of Abeta pathway
- **ApoE isoform effects**: on Abeta clearance and complement regulation
- **Endosomal cholesterol accumulation**: affects APP processing

### G. Proteostasis
- **Autophagy/lysosomal degradation**: tau and Abeta clearance
- **Proteasome/ubiquitin system**: protein quality control
- **Endolysosomal trafficking**: PICALM, BIN1, CD2AP involvement
- **TDP-43 phase transitions**: Kd = 112 nM for RNA binding; aggregation kinetics

## 3. Extractable Reactions and Rate Laws

### 3.1 Tau Phosphorylation/Dephosphorylation (from PMC6537657)

```antimony
# Tau phosphorylation at specific sites (mass action kinetics)
# Site-specific rates differ between brain (intracellular) and CSF (extracellular)

# T217 phosphorylation (fast turnover, enriched in CSF)
Tau_Brain + Kinase_Brain -> pTau_T217_Brain + Kinase_Brain ; k_phos_T217 * Tau_Brain * Kinase_Brain
pTau_T217_Brain -> Tau_Brain ; k_dephos_T217 * pTau_T217_Brain

# S202 phosphorylation (hypophosphorylated in CSF vs brain)
Tau_Brain + Kinase_Brain -> pTau_S202_Brain + Kinase_Brain ; k_phos_S202 * Tau_Brain * Kinase_Brain
pTau_S202_Brain -> Tau_Brain ; k_dephos_S202 * pTau_S202_Brain

# Rate type: MA (mass action, intracellular)
# Phosphorylation rate at S113: 0.2-0.5%
# pT217 has shorter turnover rate than pS214
```

### 3.2 Tau ISF/CSF Turnover (from PMC6410369)

```antimony
# Extracellular tau clearance (first-order)
# Half-life = 1-2 hours in ISF/CSF
Tau_ISF -> ; k_clear_tau_ISF * Tau_ISF
# k_clear_tau_ISF = ln(2) / t_half = 0.693 / 1.5 h ~ 0.46 h^-1

# Neuronal activity-dependent tau release
# Sleep-wake modulated
Tau_Neuron -> Tau_ISF ; k_release_tau * Tau_Neuron * NeuronalActivity
# Rate type: UDF (unidirectional flow)
```

### 3.3 Tau Aggregation (from PMC6555816)

```antimony
# Tau monomer -> oligomer -> fibril (nucleation-elongation)
# t1/2 ~ 7 hours for VQIVYK motif-dependent aggregation

# Nucleation (second-order)
2 TauMono_Brain -> TauOligo_Brain ; k_nuc_tau * TauMono_Brain^2
# k_nuc_tau estimated from t1/2 ~ 7 h

# Elongation (pseudo first-order)
TauMono_Brain + TauOligo_Brain -> TauOligo_Brain ; k_elong_tau * TauMono_Brain * TauOligo_Brain

# Fibril formation
TauOligo_Brain -> TauFibril_Brain ; k_fibril_tau * TauOligo_Brain

# Rate type: custom_conc_per_time
```

### 3.4 ApoE-C1q Complement Interaction (from PMC6420126)

```antimony
# ApoE binds activated C1q with high affinity
# KD ~ 500 pM = 5e-10 M

# Reversible binding (RMA)
ApoE_ISF + C1q_act_ISF -> ApoE_C1q_ISF ; kon_ApoE_C1q * ApoE_ISF * C1q_act_ISF
ApoE_C1q_ISF -> ApoE_ISF + C1q_act_ISF ; koff_ApoE_C1q * ApoE_C1q_ISF
# KD = koff/kon = 500 pM

# Complement cascade attenuation
C1q_act_ISF + C5_ISF -> C5a_ISF + C5b_ISF ; k_complement * C1q_act_ISF * C5_ISF
# ApoE sequestration of C1q reduces downstream complement activation
# Rate type: RMA (reversible mass action) for binding
```

```antimony
# Microglial phagocytosis (Michaelis-Menten / saturation kinetics)
# CD22 is negative regulator; TREM2 is positive regulator
```

```antimony
# APP cleavage by BACE1 in early endosome (pH-dependent)
```

### 3.7 Abeta-Dependent Neuronal Hyperactivation (from PMC6690382)

```antimony
# Abeta oligomers enhance glutamate release / impair reuptake
# Creating vicious cycle: more activity -> more Abeta release

# Abeta-enhanced glutamate accumulation (reduced clearance)
# Modeled as decreased reuptake rate proportional to AB_oligo
Glut_Synapse -> ; k_glut_reuptake * Glut_Synapse / (1 + AB_oligo_ISF / Ki_AB_glut)

# NMDA receptor overactivation -> calcium influx
Glut_Synapse -> Ca_Neuron ; k_NMDA * Glut_Synapse * NMDAR_open

# Activity-dependent Abeta release (positive feedback)
AB_Neuron -> AB_ISF ; k_act_release * AB_Neuron * NeuronalActivity
# Rate type: custom_conc_per_time
```

### 3.8 Capillary Stalling / CBF Reduction (from PMC6508667)

```antimony
# Neutrophil adhesion in brain capillaries
# Stalling half-life < 5 min; reduces CBF

Neutrophil_Blood + Capillary_open -> Capillary_stalled ; k_adhesion * Neutrophil_Blood * Capillary_open
Capillary_stalled -> Capillary_open + Neutrophil_Blood ; k_unstall * Capillary_stalled
# k_unstall = ln(2) / 5 min ~ 0.14 min^-1

# CBF proportional to open capillaries
# CBF = CBF_max * Capillary_open / Capillary_total
# Rate type: RMA (reversible mass action)
```

### 3.9 TDP-43 Phase Transition (from PMC6472983)

```antimony
# TDP-43 RNA binding (protective against aggregation)
# KD = 112 nM for RNA oligonucleotide binding

TDP43_soluble + RNA -> TDP43_RNA ; kon_TDP43 * TDP43_soluble * RNA
TDP43_RNA -> TDP43_soluble + RNA ; koff_TDP43 * TDP43_RNA
# KD = koff/kon = 112 nM

# TDP-43 phase separation (when unbound)
TDP43_soluble -> TDP43_condensate ; k_phase_sep * TDP43_soluble^2
TDP43_condensate -> TDP43_aggregate ; k_mature * TDP43_condensate
# Rate type: RMA for binding, custom for phase transitions
```
